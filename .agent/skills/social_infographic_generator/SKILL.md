# Social Infographic Generator

X（Twitter）とThreadsで使用できる日本語インフォグラフィック画像を一括生成します。

## 使用方法

### 基本コマンド（Threads用）
```
Piste_threads_image_prompt.mdから画像を生成して、Piste_threads_image/フォルダに保存してください。
プラットフォームはthreadsです。
```

### X用カスタム生成
```
X_image_prompt.mdから16:9の横長画像を生成して、X_images/フォルダに保存してください。
プラットフォームはxです。
```

## 実行手順

### 1. プロンプトファイルを読み込む

指定されたMarkdownファイル（例：`Piste_threads_image_prompt.md`）を読み込みます。

**対応フォーマット：**

**リスト形式：**
```markdown
- **2026-02-10 06:00**: 日本のSNS向けインフォグラフィック。テーマは「朝の痩せ習慣」...
- **2026-02-10 12:00**: 日本のSNS向けインフォグラフィック。テーマは「昼食の選び方」...
```

**セクション形式：**
```markdown
## 投稿1 (2026-02-10, 06:00)
**Prompt**:
> 日本のSNS向けインフォグラフィック...
```

### 2. 既存画像を確認

出力ディレクトリ内の既存ファイルをチェックし、既に生成済みの画像をスキップします。

**命名規則：**
- **Threads**: `2026-02-10-6:00 piste_threads.png`
- **X**: `2026-02-10-6:00 x.png`

### 3. 未生成プロンプトを抽出

まだ画像が生成されていないプロンプトのみをリストアップします。

### 4. 画像生成ループ

各プロンプトに対して以下を実行：

1. `generate_image`ツールで画像を生成
2. アーティファクトディレクトリから指定の出力ディレクトリに画像を移動
3. 正しいファイル名にリネーム（`YYYY-MM-DD-H:MM [platform].png`）

**重要な注意事項：**
- アスペクト比は**9:16（縦長）**がデフォルト
- Xプラットフォームで16:9を指定された場合は横長で生成
- 時間の「分」が00の場合はそのまま、時間が1桁の場合は0をつけない（例: 6:00）

### 5. 完了サマリーを表示

```
✅ 画像生成完了

生成枚数: 21枚
保存先: /Users/ishikawasuguru/Threads_piste/Piste_threads_image/
スキップ: 3枚（既存）

次のステップ：
- Googleドライブへアップロード: cd image_upload && python3 piste_threads_image_uplorder.py
- Notion同期: python3 piste_threads_drive_to_notion.py
```

## プロンプト形式の詳細

### 推奨形式（リスト）

```markdown
- **YYYY-MM-DD HH:MM**: [画像生成プロンプト]
```

**例：**
```markdown
- **2026-02-10 06:00**: 日本のSNS向けインフォグラフィック（図解）。テーマは「朝の痩せ習慣」。以下のポイントを視覚的に整理：1. 起きたら白湯、2. カーテンを開けて朝日、3. 朝食にプロテイン。イラスト：太陽、コップ、シェイカー。爽やかな青とオレンジ。8k --ar 9:16
```

### セクション形式

```markdown
## [タイトル] ([日付], [時刻])
**Context**: [背景情報]
**Prompt**:
> [詳細なプロンプト]
```

## ファイル命名規則

### Threads
```
2026-02-10-6:00 piste_threads.png
2026-02-11-12:00 piste_threads.png
```

### X（Twitter）
```
2026-02-10-6:00 x.png
2026-02-11-12:00 x.png
```

### 命名ルール
- 日付形式：`YYYY-MM-DD`
- 時間形式：`H:MM`（先頭のゼロなし、例：`6:00`, `18:00`）
- プラットフォーム識別子：` piste_threads` または ` x`
- 拡張子：`.png`

## アスペクト比設定

| プラットフォーム | デフォルト | カスタム可能 |
|----------------|-----------|------------|
| Threads | 9:16（縦長） | 1:1, 4:5も可 |
| X（Twitter） | 9:16（縦長） | 16:9（横長）推奨 |

## 既存ワークフローとの関係

このスキルは**独立運用**されます：

### Piste Threadsの場合
```
手動ステップ1: エージェントで投稿案・プロンプト作成
手動ステップ2: このスキルで画像生成
自動ステップ3: Pythonスクリプトでアップロード・同期
```

### X（Twitter）の場合
```
手動ステップ1: プロンプトファイルを作成
手動ステップ2: このスキルで画像生成（16:9推奨）
手動ステップ3: 手動またはスクリプトでアップロード
```

## トラブルシューティング

### 画像が生成されない
- プロンプトファイルのフォーマットを確認
- 日時形式が正しいか確認（`YYYY-MM-DD HH:MM`）

### ファイル名が正しくない
- 命名規則を確認
- 時間の先頭ゼロが含まれていないか確認

### 既存画像が上書きされる
- このスキルは既存ファイルをスキップします
- 強制的に再生成する場合は、該当ファイルを削除してから実行

```

```

```
X_images/フォルダに保存してください。プラットフォームはxです。
```
