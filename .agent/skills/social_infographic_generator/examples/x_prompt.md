# X（Twitter）用インフォグラフィックプロンプト例

このファイルはX投稿用の画像プロンプト例です。

## 共通設定
- **Aspect Ratio**: 16:9（横長）または 9:16（縦長）
- **Style**: 日本のインフォグラフィック、シンプルで読みやすい、SNS映えするデザイン
- **Text**: 簡潔なテキスト、視覚的インパクト重視

## アスペクト比の使い分け

### 16:9（横長）- 推奨
- フロー図、比較表、タイムライン
- ビジネス・テック系コンテンツ
- デスクトップ表示に最適

### 9:16（縦長）
- リスト形式、ステップバイステップ
- ライフスタイル・自己啓発系
- モバイル表示に最適

```markdown
- **YYYY-MM-DD HH:MM**: [プロンプト内容（アスペクト比を明記）]
```
